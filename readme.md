# 📄 AI-Based Resume Analyzer

An intelligent Resume Analyzer built using **Python, Machine Learning, and Streamlit** that can analyze resumes, extract key information, and predict job roles based on skills and content.

---

## 🚀 Features

- 📂 Upload Resume (PDF / TXT)
- 🧠 AI-based Job Role Prediction
- 🔍 Keyword & Skill Extraction
- 📊 Resume Scoring System
- ❌ Missing Skills Identification
- 💡 Suggestions for Improvement

---

## 🛠️ Tech Stack

- **Frontend:** Streamlit  
- **Backend:** Python  
- **Machine Learning:** Scikit-learn (TF-IDF + Naive Bayes)  
- **Libraries:** Pandas, NumPy, PyPDF2
